本指南介绍如何使用 EmDash 管理面板创建、编辑和管理内容。

## 访问管理面板

在浏览器中打开您站点的 `/_emdash/admin`。使用安装过程中创建的凭据登录。

仪表板显示：

- **侧边栏** - 导航到集合、媒体和设置
- **内容列表** - 所选集合中的条目
- **快速操作** - 创建新内容、批量操作

## 创建内容

1. 在侧边栏中点击集合名称（例如 **Posts**）

2. 点击 **New Post**（或您集合的对应按钮）

3. 填写必填字段：
    - **Title** - 内容的显示名称
    - **Slug** - URL 标识符（根据标题自动生成，可编辑）

4. 使用富文本编辑器添加内容

5. 在侧边栏中设置元数据：
    - **Status** - 草稿、已发布或已归档
    - **Publication date** - 发布时间
    - **Categories and tags** - 分类和标签分配

6. 点击 **Save**

<Aside>
	草稿仅在管理面板可见。将状态更改为 **Published** 可使内容在您的网站上可见。
</Aside>

## 内容状态

每个条目都有以下三种状态之一：

| 状态           | 可见性       | 用例             |
| -------------- | ------------ | ---------------- |
| **Draft**      | 仅管理面板   | 进行中的工作     |
| **Published**  | 公开         | 已发布内容       |
| **Archived**   | 仅管理面板   | 已归档内容       |

使用编辑器侧边栏中的下拉菜单更改状态。

## 富文本编辑器

EmDash 的编辑器支持：

- **标题** - H2 到 H6
- **格式** - 粗体、斜体、下划线、删除线
- **列表** - 有序和无序列表
- **链接** - 内部和外部链接
- **图片** - 从媒体库插入
- **代码块** - 支持语法高亮
- **嵌入** - YouTube、Vimeo、Twitter
- **区块** - 通过 `/section` 命令插入可重用的内容区块

### 斜杠命令

输入 `/` 以访问快速插入命令：

| 命令                       | 操作                               |
| -------------------------- | ---------------------------------- |
| `/section`                 | 插入一个可重用的区块               |
| `/image`                   | 从媒体库插入图片                   |
| `/code`                    | 插入一个代码块                     |

### 键盘快捷键

| 操作     | 快捷键                     |
| -------- | -------------------------- |
| 粗体     | `Ctrl/Cmd + B`             |
| 斜体     | `Ctrl/Cmd + I`             |
| 链接     | `Ctrl/Cmd + K`             |
| 撤销     | `Ctrl/Cmd + Z`             |
| 重做     | `Ctrl/Cmd + Shift + Z`     |
| 保存     | `Ctrl/Cmd + S`             |

### 插入图片

1. 点击工具栏中的图片按钮

2. 从媒体库中选择现有图片，或上传新图片

3. 添加替代文本（为无障碍访问所必需）

4. 调整对齐方式和大小选项

5. 点击 **Insert**

## 编辑内容

1. 导航到包含该内容的集合

2. 点击您要编辑的条目

3. 进行更改

4. 点击 **Save**

对已发布内容的更改会立即在您的网站上生效。无需重新构建。

### 修订历史

EmDash 会跟踪内容的更改。从编辑器侧边栏访问修订历史：

1. 在编辑器侧边栏中点击 **Revisions**

2. 查看带有时间戳的先前版本列表

3. 点击某个修订版进行预览

4. 点击 **Restore** 以恢复到该版本

<Aside type="caution">
	恢复修订版会创建一个包含恢复内容的新修订版。原始修订历史将被保留。
</Aside>

## 批量操作

同时对多个条目执行操作：

1. 使用复选框在内容列表中选择条目

2. 点击 **Bulk Actions** 下拉菜单

3. 选择一个操作：
    - **Publish** - 将所有选中的条目设置为已发布
    - **Archive** - 将所有选中的条目设置为已归档
    - **Delete** - 永久删除选中的条目

4. 确认操作

## 搜索和筛选

### 搜索

使用搜索框按标题或内容查找内容。搜索不区分大小写，并匹配部分单词。

### 筛选器

按以下条件筛选内容列表：

- **Status** - 草稿、已发布、已归档
- **Date range** - 创建或修改日期范围
- **Author** - 内容创建者
- **Taxonomy** - 分类或标签分配

点击 **Clear Filters** 重置。

## 定时发布内容

安排内容在将来某个日期发布：

1. 创建或编辑内容

2. 将状态设置为 **Draft**

3. 将 **Publication date** 设置为将来的日期和时间

4. 点击 **Save**

当发布日期到达时，内容将自动变为已发布状态。

<Aside type="tip">
	定时发布功能要求您的站点是服务器端渲染的，或者有一个检查待发布内容的计划任务。
</Aside>

## 删除内容

从编辑界面或内容列表中删除内容：

### 从编辑器删除

1. 打开您要删除的内容

2. 点击工具栏中的 **Delete**

3. 确认删除

### 从列表删除

1. 使用复选框选择条目

2. 点击 **Bulk Actions** > **Delete**

3. 确认删除

<Aside type="caution">
	删除的内容将被永久移除，无法恢复。如果您以后可能需要该内容，请考虑使用归档功能。
</Aside>

## 内容 API

如需以编程方式访问，请使用 EmDash 管理 API：

### 创建内容

```bash
POST /_emdash/api/content/posts
Content-Type: application/json
Authorization: Bearer YOUR_API_TOKEN

{
  "title": "My New Post",
  "slug": "my-new-post",
  "content": "<p>Post content here</p>",
  "status": "draft"
}
```

### 更新内容

```bash
PUT /_emdash/api/content/posts/my-new-post
Content-Type: application/json
Authorization: Bearer YOUR_API_TOKEN

{
  "title": "Updated Title",
  "status": "published"
}
```

### 删除内容

```bash
DELETE /_emdash/api/content/posts/my-new-post
Authorization: Bearer YOUR_API_TOKEN
```

## 翻译内容

当 [i18n 功能启用](/zh-cn/guides/internationalization/) 时，您可以创建任何内容条目的翻译。

### 创建翻译

1. 打开您要翻译的内容条目

2. 在编辑器侧边栏中，找到 **Translations** 面板

3. 点击目标语言环境旁边的 **Translate**

4. 编辑预填充的内容 — 为新语言调整标题、slug 和正文

5. 点击 **Save**

新的翻译链接到原始条目，并作为草稿开始。翻译准备就绪后，可以独立发布它。

### 在翻译之间切换

Translations 面板显示所有已配置的语言环境。点击任何现有翻译旁边的 **Edit** 可直接导航到该翻译。当前语言环境会用一个对勾标记。

### 语言环境筛选器

在内容列表中，使用工具栏中的语言环境下拉菜单按语言筛选条目。每个条目在专用列中显示其语言环境。

<Aside type="tip">
  每个翻译都有自己的 slug、状态和修订历史。可以独立发布、安排和管理翻译。
</Aside>

有关配置、查询和语言切换器的完整详细信息，请参阅 [国际化指南](/zh-cn/guides/internationalization/)。
