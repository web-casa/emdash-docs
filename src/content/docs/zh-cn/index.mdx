## 快速开始

```bash
# Create a new EmDash site
npm create astro@latest -- --template @emdash-cms/template-blog

# Start the dev server
npm run dev

# Visit the admin at http://localhost:4321/_emdash/admin
```

## 关于 EmDash

- **Astro 原生 CMS** — 不是后加 Astro 适配器的通用 CMS，而是为 Astro 设计
- **类型安全内容模型** — 结构化内容和 TypeScript 支持更容易协作和维护
- **可扩展架构** — 支持插件、钩子、管理界面扩展和灵活部署
- **兼顾 WordPress 使用习惯** — 菜单、小部件、分类法和迁移场景都考虑在内
