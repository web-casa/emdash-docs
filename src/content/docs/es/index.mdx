## Inicio Rápido

```bash
# Crear un nuevo sitio con EmDash
npm create astro@latest -- --template @emdash-cms/template-blog

# Iniciar el servidor de desarrollo
npm run dev

# Abrir el panel en http://localhost:4321/_emdash/admin
```

## Acerca de EmDash

- **CMS Nativo para Astro** — Construido para Astro en lugar de adaptado a él posteriormente
- **Contenido con Tipos Seguros** — Modelos de contenido estructurado y soporte de TypeScript generado
- **Arquitectura Extensible** — Complementos, ganchos, extensiones de la interfaz de administración y flexibilidad de despliegue
- **Conceptos Amigables para WordPress** — Menús, widgets, taxonomías y flujos de trabajo orientados a la migración
