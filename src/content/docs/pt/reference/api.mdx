### `getEmDashCollection()`


#### ParĂ˘metros

| ParĂ˘metro    | Tipo               | DescriĂ§ĂŁo                |

#### Retorna


#### Exemplos

```ts

	status: "published",

	status: "published",
	where: { category: "news" },

// Lidar com erros

	return new Response("Server error", { status: 500 });
```


## Tratamento de Erros

A EmDash exporta classes de erro para lidar com falhas especĂ­ficas:

```ts

  EmDashDatabaseError,
  EmDashValidationError,
  EmDashStorageError,
  SchemaError,
} from "emdash";

  await repo.create({ ... });
} catch (error) {
  if (error instanceof EmDashValidationError) {
    console.error("Validation failed:", error.message);
  }
  if (error instanceof SchemaError) {
    console.error("Schema error:", error.code, error.details);
  }
}
```
