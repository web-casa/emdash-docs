## Início Rápido

```bash
# Create a new EmDash site
npm create astro@latest -- --template @emdash-cms/template-blog

# Inicie o servidor de desenvolvimento
npm run dev

# Acesse o admin em http://localhost:4321/_emdash/admin
```

## Sobre o EmDash

- **CMS Nativo para Astro** — Construído para o Astro em vez de adaptado a ele posteriormente
- **Conteúdo com Tipagem Segura** — Modelos de conteúdo estruturados e suporte a TypeScript gerado
- **Arquitetura Extensível** — Plugins, hooks, extensões de interface de administração e flexibilidade de implantação
- **Conceitos Amigáveis ao WordPress** — Menus, widgets, taxonomias e fluxos de trabalho orientados à migração
