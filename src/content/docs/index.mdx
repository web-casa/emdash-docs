## Quick Start

```bash
# Create a new EmDash site
npm create astro@latest -- --template @emdash-cms/template-blog

# Start the dev server
npm run dev

# Visit the admin at http://localhost:4321/_emdash/admin
```

## About EmDash

- **Astro-native CMS** — Built for Astro instead of adapted to it later
- **Type-safe content** — Structured content models and generated TypeScript support
- **Extensible architecture** — Plugins, hooks, admin UI extensions, and deployment flexibility
- **WordPress-friendly concepts** — Menus, widgets, taxonomies, and migration-oriented workflows
