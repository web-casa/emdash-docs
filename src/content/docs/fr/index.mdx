## Démarrage Rapide

```bash
# Create a new EmDash site
npm create astro@latest -- --template @emdash-cms/template-blog

# Démarrer le serveur de développement
npm run dev

# Visitez l'administration à http://localhost:4321/_emdash/admin
```

## À Propos d'EmDash

- **CMS Natif Astro** — Construit pour Astro au lieu d'y être adapté ultérieurement
- **Contenu typé** — Modèles de contenu structurés et support TypeScript généré
- **Architecture extensible** — Plugins, hooks, extensions de l'interface d'administration et flexibilité de déploiement
- **Concepts compatibles WordPress** — Menus, widgets, taxonomies et flux de travail orientés migration
