Les taxonomies sont des systèmes de classification pour organiser le contenu. EmDash inclut des catégories et des étiquettes intégrées, et prend en charge les taxonomies personnalisées pour des besoins de classification spécialisés.

## Taxonomies intégrées

EmDash fournit deux taxonomies par défaut :

| Taxonomie      | Type         | Description                                                                 |
| -------------- | ------------ | --------------------------------------------------------------------------- |
| **Catégories** | Hiérarchique | Classification imbriquée avec des relations parent-enfant                   |
| **Étiquettes** | Plate        | Libellés simples sans hiérarchie                                            |

Les deux sont disponibles pour la collection d'articles par défaut.

## Gestion des termes

### Créer un terme

<Tabs>
  <TabItem label="Tableau de bord d'administration">

1. Allez sur la page de la taxonomie (par exemple, `/_emdash/admin/taxonomies/category`)

2. Saisissez le nom du terme dans le formulaire **Ajouter**

3. Optionnellement, définissez :
   - **Slug** - Identifiant d'URL (généré automatiquement à partir du nom)
   - **Parent** - Pour les taxonomies hiérarchiques
   - **Description** - Description du terme

4. Cliquez sur **Ajouter**

  </TabItem>
  <TabItem label="Éditeur de contenu">

5. Ouvrez une entrée de contenu dans l'éditeur

6. Trouvez le panneau de taxonomie dans la barre latérale

7. Pour les catégories : cochez les cases des termes applicables, ou cliquez sur **+ Ajouter**

8. Pour les étiquettes : saisissez les noms des étiquettes séparés par des virgules

9. Enregistrez le contenu

  </TabItem>
  <TabItem label="API">

```bash
POST /_emdash/api/taxonomies/category/terms
Content-Type: application/json
Authorization: Bearer YOUR_API_TOKEN

{
  "slug": "tutorials",
  "label": "Tutoriels",
  "parentId": "term_abc",
  "description": "Guides pratiques et tutoriels"
}
```

  </TabItem>
</Tabs>

### Modifier un terme

1. Allez sur la page des termes de la taxonomie

2. Cliquez sur **Modifier** à côté du terme

3. Mettez à jour le nom, le slug, le parent ou la description

4. Cliquez sur **Enregistrer**

### Supprimer un terme

1. Allez sur la page des termes de la taxonomie

2. Cliquez sur **Supprimer** à côté du terme

3. Confirmez la suppression

<Aside type="caution">
	Supprimer un terme le retire de tout le contenu associé. Le contenu n'est pas supprimé, seule l'attribution du terme
	est retirée.
</Aside>

## Interroger les taxonomies

EmDash fournit des fonctions pour interroger les termes de taxonomie et filtrer le contenu par terme.

### Obtenir tous les termes

Récupérer tous les termes d'une taxonomie :

```ts
import { getTaxonomyTerms } from "emdash";

// Obtenir toutes les catégories (retourne une structure arborescente)
const categories = await getTaxonomyTerms("category");

// Obtenir toutes les étiquettes (retourne une liste plate)
const tags = await getTaxonomyTerms("tag");
```

Pour les taxonomies hiérarchiques, les termes incluent un tableau `children` :

```ts
interface TaxonomyTerm {
	id: string;
	name: string; // Taxonomy name ("category")
	slug: string; // Term slug ("news")
	label: string; // Display label ("News")
	parentId?: string;
	description?: string;
	children: TaxonomyTerm[];
	count?: number; // Number of entries with this term
}
```

### Obtenir un terme unique

```ts
import { getTerm } from "emdash";

const category = await getTerm("category", "news");
// Retourne TaxonomyTerm ou null
```

### Obtenir les termes pour une entrée

```ts
import { getEntryTerms } from "emdash";

// Obtenir toutes les catégories pour un article
const categories = await getEntryTerms("posts", "post-123", "category");

// Obtenir toutes les étiquettes pour un article
const tags = await getEntryTerms("posts", "post-123", "tag");
```

### Filtrer le contenu par terme

Utilisez `getEmDashCollection` avec le filtre `where` :

```ts
import { getEmDashCollection } from "emdash";

// Articles dans la catégorie "news"
const { entries: newsPosts } = await getEmDashCollection("posts", {
	status: "published",
	where: { category: "news" },
});

// Articles avec l'étiquette "javascript"
const { entries: jsPosts } = await getEmDashCollection("posts", {
	status: "published",
	where: { tag: "javascript" },
});
```

Ou utilisez la fonction de commodité :

```ts
import { getEntriesByTerm } from "emdash";

const newsPosts = await getEntriesByTerm("posts", "category", "news");
```

## Création de pages de taxonomie

### Archive de catégorie

Créez une page qui liste les articles d'une catégorie :

```
astro title="src/pages/category/[slug].astro"
---
import { getTaxonomyTerms, getTerm, getEmDashCollection } from "emdash";
import Base from "../../../layouts/Base.astro";

export async function getStaticPaths() {
  const categories = await getTaxonomyTerms("category");

  // Aplatir l'arbre hiérarchique pour le routage
  function flatten(terms) {
    return terms.flatMap((term) => [term, ...flatten(term.children)]);
  }

  return flatten(categories).map((cat) => ({
    params: { slug: cat.slug },
    props: { category: cat },
  }));
}

const { category } = Astro.props;

const { entries: posts } = await getEmDashCollection("posts", {
  status: "published",
  where: { category: category.slug },
});
---

<Base title={category.label}>
  <h1>{category.label}</h1>
  {category.description && <p>{category.description}</p>
}
  <p>{category.count} articles</p>

  <ul>
    {posts.map((post) => (
      <li>
        <a href={`/blog/${post.data.slug}`}>{post.data.title}</a>
      </li>
    ))}
  </ul>
</Base>
```

### Archive d'étiquettes

Créez une page qui liste les articles avec une étiquette :

```
astro title="src/pages/tag/[slug].astro"
---
import { getTaxonomyTerms, getEmDashCollection } from "emdash";
import Base from "../../../layouts/Base.astro";

export async function getStaticPaths() {
  const tags = await getTaxonomyTerms("tag");

  return tags.map((tag) => ({
    params: { slug: tag.slug },
    props: { tag },
  }));
}

const { tag } = Astro.props;

const { entries: posts } = await getEmDashCollection("posts", {
  status: "published",
  where: { tag: tag.slug },
});
---

<Base title={`Articles étiquetés "${tag.label}"`}>
  <h1>#{tag.label}</h1>

  <ul>
    {posts.map((post) => (
      <li>
        <a href={`/blog/${post.data.slug}`}>{post.data.title}</a>
      </li>
    ))}
  </ul>
</Base>
```

### Widget de liste de catégories

Affichez une liste de catégories avec le nombre d'articles :

```
astro title="src/components/CategoryList.astro"
---
import { getTaxonomyTerms } from "emdash";

const categories = await getTaxonomyTerms("category");
---

<nav class="category-list">
  <h3>Catégories</h3>
  <ul>
    {categories.map((cat) => (
      <li>
        <a href={`/category/${cat.slug}`}>
          {cat.label} ({cat.count})
        </a>
        {cat.children.length > 0 && (
          <ul>
            {cat.children.map((child) => (
              <li>
                <a href={`/category/${child.slug}`}>
                  {child.label} ({child.count})
                </a>
              </li>
            ))}
          </ul>
        )}
      </li>
    ))}
  </ul>
</nav>
```

### Nuage d'étiquettes

Affichez les étiquettes avec une taille basée sur l'utilisation :

```
astro title="src/components/TagCloud.astro"
---
import { getTaxonomyTerms } from "emdash";

const tags = await getTaxonomyTerms("tag");

// Calculer les tailles de police basées sur le nombre
const counts = tags.map((t) => t.count ?? 0);
const maxCount = Math.max(...counts, 1);
const minSize = 0.8;
const maxSize = 2;

function getSize(count: number) {
  const ratio = count / maxCount;
  return minSize + ratio * (maxSize - minSize);
}
---

<div class="tag-cloud">
  {tags.map((tag) => (
    <a
      href={`/tag/${tag.slug}`}
      style={`font-size: ${getSize(tag.count ?? 0)}rem`}
    >
      {tag.label}
    </a>
  ))}
</div>
```

## Affichage des termes sur le contenu

Affichez les catégories et étiquettes sur un article :

```
astro title="src/components/PostTerms.astro"
---
import { getEntryTerms } from "emdash";

interface Props {
  collection: string;
  entryId: string;
}

const { collection, entryId } = Astro.props;

const categories = await getEntryTerms(collection, entryId, "category");
const tags = await getEntryTerms(collection, entryId, "tag");
---

<div class="post-terms">
  {categories.length > 0 && (
    <div class="categories">
      <span>Publié dans :</span>
      {categories.map((cat, i) => (
        <>
          {i > 0 && ", "}
          <a href={`/category/${cat.slug}`}>{cat.label}</a>
        </>
      ))}
    </div>
  )}

  {tags.length > 0 && (
    <div class="tags">
      {tags.map((tag) => (
        <a href={`/tag/${tag.slug}`} class="tag">
          #{tag.label}
        </a>
      ))}
    </div>
  )}
</div>
```

## Taxonomies Personnalisées

Créez des taxonomies au-delà des catégories et des étiquettes pour des besoins spécialisés.

### Créer une Taxonomie Personnalisée

Utilisez l'API d'administration pour créer une taxonomie :

```bash
POST /_emdash/api/taxonomies
Content-Type: application/json
Authorization: Bearer YOUR_API_TOKEN

{
  "name": "genre",
  "label": "Genres",
  "labelSingular": "Genre",
  "hierarchical": true,
  "collections": ["books", "movies"]
}
```

### Utiliser des Taxonomies Personnalisées

Interrogez et affichez les taxonomies personnalisées de la même manière que celles intégrées :

```ts
import { getTaxonomyTerms, getEmDashCollection } from "emdash";

// Obtenir tous les genres
const genres = await getTaxonomyTerms("genre");

// Obtenir les livres d'un genre
const { entries: sciFiBooks } = await getEmDashCollection("books", {
	where: { genre: "science-fiction" },
});
```

### Assigner aux Collections

Les taxonomies spécifient à quelles collections elles s'appliquent :

```ts
{
  "name": "difficulty",
  "label": "Difficulty Levels",
  "hierarchical": false,
  "collections": ["recipes", "tutorials"]
}
```

## Référence de l'API des Taxonomies

### Points de Terminaison REST

| Point de Terminaison                           | Méthode | Description               |
| --------------------------------------------- | ------ | ------------------------- |
| `/_emdash/api/taxonomies`                   | GET    | Lister les définitions de taxonomies |
| `/_emdash/api/taxonomies`                   | POST   | Créer une taxonomie       |
| `/_emdash/api/taxonomies/:name/terms`       | GET    | Lister les termes         |
| `/_emdash/api/taxonomies/:name/terms`       | POST   | Créer un terme            |
| `/_emdash/api/taxonomies/:name/terms/:slug` | GET    | Obtenir un terme          |
| `/_emdash/api/taxonomies/:name/terms/:slug` | PUT    | Mettre à jour un terme    |
| `/_emdash/api/taxonomies/:name/terms/:slug` | DELETE | Supprimer un terme        |

### Assigner des Termes au Contenu

```bash
POST /_emdash/api/content/posts/post-123/terms/category
Content-Type: application/json
Authorization: Bearer YOUR_API_TOKEN

{
  "termIds": ["term_news", "term_featured"]
}
```
