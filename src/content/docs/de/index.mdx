## Schnellstart

```bash
# Neue EmDash-Website erstellen
npm create astro@latest -- --template @emdash-cms/template-blog

# Starten Sie den Dev-Server
npm run dev

# Öffnen Sie die Admin-Oberfläche unter http://localhost:4321/_emdash/admin
```

## Über EmDash

- **Astro-natives CMS** — Für Astro entwickelt, statt später daran angepasst
- **Typsichere Inhalte** — Strukturierte Inhaltsmodelle und generierte TypeScript-Unterstützung
- **Erweiterbare Architektur** — Plugins, Hooks, Admin-UI-Erweiterungen und Bereitstellungsflexibilität
- **WordPress-freundliche Konzepte** — Menüs, Widgets, Taxonomien und migrationsorientierte Workflows
