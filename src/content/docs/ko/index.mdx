- **시작하기 가이드** — 새로운 EmDash 사용자를 위한 명확한 설정 지침
- **실용적인 FAQ** — 공개 프로젝트 토론과 일반적인 어려움에서 수집된 추가 답변

## 빠른 시작

```bash
# Create a new EmDash site
npm create astro@latest -- --template @emdash-cms/template-blog

# 개발 서버 시작하기
npm run dev

# 관리자 페이지 방문하기 http://localhost:4321/_emdash/admin
```

## EmDash 소개

- **Astro 네이티브 CMS** — 나중에 적응하는 것이 아닌 Astro를 위해 구축됨
- **타입 안전 콘텐츠** — 구조화된 콘텐츠 모델과 생성된 TypeScript 지원
- **확장 가능한 아키텍처** — 플러그인, 훅, 관리자 UI 확장 및 배포 유연성
- **WordPress 친화적 개념** — 메뉴, 위젯, 분류 체계 및 마이그레이션 지향 워크플로우
