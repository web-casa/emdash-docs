EmDash는 콘텐츠 관리, 미디어 업로드 및 스키마 작업을 위한 REST API를 `/_emdash/api/`에서 제공합니다.

## 인증

API 요청에는 Bearer 토큰을 통한 인증이 필요합니다:

```http
Authorization: Bearer <token>
```

토큰은 관리자 인터페이스 또는 프로그래밍 방식으로 생성할 수 있습니다.

## 응답 형식

모든 응답은 일관된 형식을 따릅니다:

```json
// Success
{
  "success": true,
  "data": { ... }
}

// 오류
{
  "success": false,
  "error": {
    "code": "ERROR_CODE",
    "message": "사람이 읽을 수 있는 메시지",
    "details": { ... }
  }
}
```

## 콘텐츠 엔드포인트

### 콘텐츠 목록 조회

```http
GET /_emdash/api/content/:collection
```

#### 매개변수

| 매개변수    | 유형       | 설명                                      |
| ------------ | -------- | --------------------------------------- |
| `collection` | `string` | 컬렉션 슬러그 (경로)                     |
| `cursor`     | `string` | 페이지네이션 커서 (쿼리)                  |
| `limit`      | `number` | 페이지당 항목 수 (쿼리, 기본값: 50)       |
| `status`     | `string` | 상태별 필터 (쿼리)                        |
| `orderBy`    | `string` | 정렬 기준 필드 (쿼리)                     |
| `order`      | `string` | 정렬 방향: `asc` 또는 `desc` (쿼리)       |

#### 응답

```json
{
  "success": true,
  "data": {
    "items": [
      {
        "id": "01HXK5MZSN...",
        "type": "posts",
        "slug": "hello-world",
        "data": { "title": "안녕하세요", ... },
        "status": "published",
        "createdAt": "2025-01-24T12:00:00Z",
        "updatedAt": "2025-01-24T12:00:00Z"
      }
    ],
    "nextCursor": "eyJpZCI6..."
  }
}
```

### 콘텐츠 조회

```http
GET /_emdash/api/content/:collection/:id
```

#### 응답

```json
{
  "success": true,
  "data": {
    "item": {
      "id": "01HXK5MZSN...",
      "type": "posts",
      "slug": "hello-world",
      "data": { "title": "안녕하세요", ... },
      "status": "published",
      "createdAt": "2025-01-24T12:00:00Z",
      "updatedAt": "2025-01-24T12:00:00Z"
    }
  }
}
```

### 콘텐츠 생성

```http
POST /_emdash/api/content/:collection
Content-Type: application/json
```

#### 요청 본문

```json
{
  "data": {
    "title": "New Post",
    "content": [...]
  },
  "slug": "new-post",
  "status": "draft"
}
```

#### 응답

```json
{
  "success": true,
  "data": {
    "item": { ... }
  }
}
```

### 콘텐츠 업데이트

```http
PUT /_emdash/api/content/:collection/:id
Content-Type: application/json
```

#### 요청 본문

```json
{
	"data": {
		"title": "Updated Title"
	},
	"status": "published"
}
```

### 콘텐츠 삭제

```http
DELETE /_emdash/api/content/:collection/:id
```

#### 응답

```json
{
	"success": true,
	"data": {
		"success": true
	}
}
```

## 미디어 엔드포인트

### 미디어 목록 조회

```http
GET /_emdash/api/media
```

#### 매개변수

| 매개변수  | 유형       | 설명                           |
| ---------- | -------- | ---------------------------- |
| `cursor`   | `string` | 페이지네이션 커서              |
| `limit`    | `number` | 페이지당 항목 수 (기본값: 20)  |
| `mimeType` | `string` | MIME 유형 접두사로 필터링      |

#### 응답

```json
{
	"success": true,
	"data": {
		"items": [
			{
				"id": "01HXK5MZSN...",
				"filename": "photo.jpg",
				"mimeType": "image/jpeg",
				"size": 102400,
				"width": 1920,
				"height": 1080,
				"url": "https://cdn.example.com/photo.jpg",
				"createdAt": "2025-01-24T12:00:00Z"
			}
		],
		"nextCursor": "eyJpZCI6..."
	}
}
```

### 미디어 조회

```http
GET /_emdash/api/media/:id
```

### 미디어 생성

```http
POST /_emdash/api/media
Content-Type: application/json
```

#### 요청 본문

```json
{
	"filename": "photo.jpg",
	"mimeType": "image/jpeg",
	"size": 102400,
	"width": 1920,
	"height": 1080,
	"storageKey": "uploads/photo.jpg"
}
```

<Aside>
	이 엔드포인트는 업로드 후 미디어 메타데이터를 기록합니다. 실제 파일을 업로드하려면 업로드 엔드포인트 또는 서명된 URL을 사용하세요.
</Aside>

### 미디어 업데이트

```http
PUT /_emdash/api/media/:id
Content-Type: application/json
```

#### 요청 본문

```json
{
	"alt": "Photo description",
	"caption": "Photo caption"
}
```

### 미디어 삭제

```http
DELETE /_emdash/api/media/:id
```

### 미디어 파일 조회

```http
GET /_emdash/api/media/file/:key
```

실제 파일 내용을 제공합니다. 로컬 저장소 전용입니다.

## 오류 코드

| 코드                     | HTTP 상태 | 설명                          |
| ---------------------- | ----------- | ------------------------ |
| `NOT_FOUND`            | 404         | 리소스를 찾을 수 없음      |
| `VALIDATION_ERROR`     | 400         | 잘못된 입력 데이터         |
| `UNAUTHORIZED`         | 401         | 토큰이 없거나 유효하지 않음 |
| `FORBIDDEN`            | 403         | 권한 부족                   |
| `CONTENT_LIST_ERROR`   | 500         | 콘텐츠 목록 조회 실패       |
| `CONTENT_CREATE_ERROR` | 500         | 콘텐츠 생성 실패            |
| `CONTENT_UPDATE_ERROR` | 500         | 콘텐츠 업데이트 실패        |
| `CONTENT_DELETE_ERROR` | 500         | 콘텐츠 삭제 실패            |
| `MEDIA_LIST_ERROR`     | 500         | 미디어 목록 조회 실패       |
| `MEDIA_CREATE_ERROR`   | 500         | 미디어 생성 실패            |
| `SCHEMA_ERROR`         | 400         | 스키마 작업 실패            |
| `DUPLICATE_SLUG`       | 409         | 슬러그가 이미 존재함        |
| `RESERVED_SLUG`        | 400         | 슬러그가 예약됨              |

## 검색 엔드포인트

### 전역 검색

```http
GET /_emdash/api/search?q=hello+world
```

#### 매개변수

| 매개변수     | 유형       | 설명                                      |
| ------------- | -------- | -------------------------------- |
| `q`           | `string` | 검색어 (필수)                     |
| `collections` | `string` | 쉼표로 구분된 컬렉션 슬러그        |
| `status`      | `string` | 상태별 필터 (기본값: published)    |
| `limit`       | `number` | 최대 결과 수 (기본값: 20)          |
| `cursor`      | `string` | 페이지네이션 커서                  |

#### 응답

```json
{
  "results": [
    {
      "collection": "posts",
      "id": "01HXK5MZSN...",
      "slug": "hello-world",
      "title": "안녕하세요",
      "snippet": "...this is a <mark>hello</mark>
 <mark>world</mark>
 example...",
      "score": 0.95
    }
  ],
  "nextCursor": "eyJvZmZzZXQiOjIwfQ"
}
```

### 검색 제안

```http
GET /_emdash/api/search/suggest?q=hel&limit=5
```

자동 완성을 위한 접두사 일치 제목을 반환합니다.

### 검색 인덱스 재구축

```http
POST /_emdash/api/search/rebuild
```

모든 또는 특정 컬렉션에 대한 FTS 인덱스를 재구축합니다.

### 검색 통계

```http
GET /_emdash/api/search/stats
```

컬렉션별 인덱싱된 문서 수를 반환합니다.

## 인증 엔드포인트

### 설정 상태

```http

```

### 패스키 로그인

```http
POST /_emdash/api/auth/passkey/options
```

WebAuthn 인증 옵션을 가져옵니다.

```http
POST /_emdash/api/auth/passkey/verify
Content-Type: application/json

{
  "id": "credential-id",
  "rawId": "...",
  "response": {...},
  "type": "public-key"
}
```

패스키를 확인하고 세션을 생성합니다.

### 매직 링크

```http
POST /_emdash/api/auth/magic-link/send
Content-Type: application/json

{
  "email": "user@example.com"
}
```

```http
GET /_emdash/api/auth/magic-link/verify?token=xxx
```

### 로그아웃

```http
POST /_emdash/api/auth/logout
```

### 현재 사용자

```http
GET /_emdash/api/auth/me
```

### 사용자 초대

```http
POST /_emdash/api/auth/invite
Content-Type: application/json

{
  "email": "newuser@example.com",
  "role": 30
}
```

### 패스키 관리

```http
GET /_emdash/api/auth/passkey
```

사용자의 패스키 목록을 조회합니다.

```http
POST /_emdash/api/auth/passkey/register/options
POST /_emdash/api/auth/passkey/register/verify
```

새로운 패스키를 등록합니다.

```http
PATCH /_emdash/api/auth/passkey/:id
Content-Type: application/json

{
  "name": "MacBook Pro"
}
```

패스키 이름을 변경합니다.

```http
DELETE /_emdash/api/auth/passkey/:id
```

패스키를 삭제합니다.

## 임포트 엔드포인트

### WordPress 내보내기 분석

```http
POST /_emdash/api/import/wordpress/analyze
Content-Type: multipart/form-data

file: <WXR file>
```

### WordPress 임포트 실행

```http
POST /_emdash/api/import/wordpress/execute
Content-Type: application/json

{
  "analysisId": "...",
  "options": {
    "includeMedia": true,
    "includeTaxonomies": true,
    "includeMenus": true
  }
}
```

## 요청 제한

API 엔드포인트는 배포 구성에 따라 요청 제한이 적용될 수 있습니다. 제한이 적용되면 응답에 다음이 포함됩니다:

```http
HTTP/1.1 429 Too Many Requests
Retry-After: 60
```

## CORS

이 API는 브라우저 요청을 위한 CORS를 지원합니다. 배포 시 허용된 출처를 구성하세요.
