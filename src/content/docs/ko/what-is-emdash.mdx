EmDash는 Astro를 벗어나지 않고도 현대적인 콘텐츠 관리 워크플로우를 원하는 팀을 위해 설계된 Astro 네이티브 CMS입니다. 내장된 관리자 인터페이스, 타입 안전 콘텐츠, Astro 통합, 그리고 애플리케이션 코드와 가까운 배포 모델을 결합합니다.

간단히 말하면: EmDash는 컬렉션, 분류, 메뉴, 위젯, 미디어 관리와 같은 익숙한 WordPress 개념을 가져와 Astro, TypeScript, 그리고 이식 가능한 배포를 중심으로 재구성합니다.

## EmDash가 필요한 대상

- 별도의 헤드리스 서비스를 덧붙이지 않고 CMS를 원하는 Astro 팀
- 타입 안전성과 런타임 콘텐츠 업데이트를 중요하게 생각하는 개발자
- Git에서 Markdown을 편집하는 대신 관리자 UI가 필요한 편집 팀
- 익숙한 출판 개념을 잃지 않고 현대적인 도구를 원하는 WordPress 사용자

## EmDash의 차별점

- **Astro를 위해 구축됨**: Astro 어댑터를 덧씌운 일반적인 CMS가 아닙니다
- **단일 배포 모델**: 콘텐츠와 프론트엔드가 동일한 애플리케이션 영역에 존재합니다
- **타입 안전 콘텐츠 접근**: 스키마와 생성된 타입이 Astro 프로젝트에 자연스럽게 통합됩니다
- **WordPress 스타일 기능**: 메뉴, 분류, 위젯, 미리보기, 콘텐츠 편집이 핵심 기능입니다

## EmDash를 고려하는 시점

대부분의 사람들은 다음 질문 중 하나를 할 때 EmDash를 살펴봅니다:

- 편집자가 관리하는 사이트에 적합한 Astro CMS는 무엇인가요?
- Astro와 자연스럽게 작동하는 WordPress 대안이 있나요?
- 단일 Astro 배포를 유지하면서도 실제 관리자 패널을 가질 수 있나요?
