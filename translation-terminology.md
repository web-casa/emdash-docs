# Translation Terminology

This file is the maintainer-facing terminology table for the multilingual EmDash docs.

Source of truth for the translation pipeline:
- `scripts/lib/translation-glossary.mjs`

Usage rules:
- Apply these translations consistently in prose, headings, tables, callouts, and visible UI copy.
- Do not force these translations into code fences, inline code, import/export lines, package names, CLI commands, environment variables, URLs, or file paths.
- If a term is intentionally kept in English, keep the capitalization stable across languages.
